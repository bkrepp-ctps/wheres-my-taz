# wheres-my-taz

"Where's my TAZ?" demo app.  
Uses MassGIS's Geocoding REST API  
Documentation: https://wiki.state.ma.us/display/massgis/ArcGIS+Server+-+Geocoding+-+Census+TIGER+2010  
Author: Ben Krepp

## Libraries Required
* jQUery version 1.11.2
* OpenLayers version 5.3.0
